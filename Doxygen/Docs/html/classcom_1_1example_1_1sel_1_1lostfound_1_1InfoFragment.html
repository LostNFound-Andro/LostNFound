Detailed Description

A class implementing Help and FAQ fragment of navigation bar.

Created by Rohit G on 4/2/2016. A simple Fragment subclass. Activities that contain this fragment must implement the InfoFragment.OnFragmentInteractionListener interface to handle interaction events. Use the InfoFragment#newInstance factory method to create an instance of this fragment.
