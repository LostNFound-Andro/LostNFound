Detailed Description

A class implementing fragment of feed when its clicked on navigation bar.

A simple Fragment subclass. Activities that contain this fragment must implement the FeedFragment.OnFragmentInteractionListener interface to handle interaction events. Use the FeedFragment#newInstance factory method to create an instance of this fragment.
